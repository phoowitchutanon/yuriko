# Yuriko


## Projects in this monorepo

| Folder                  | Version      | Description                          |
| ----------------------- | ------------ | ------------------------------------ |
| [client](./apps/client) | 1.0.0        | Frontend for the user.               |
| [server](./apps/server) | 1.0.0        | Backend for [client](./apps/client). |

## Setup

Install dependencies:

```bash
pnpm i
```

## Development

```bash
pnpm dev

# client only
pnpm dev:client

# server only
pnpm dev:server
```

## Production

```bash
yarn build

yarn start
```
